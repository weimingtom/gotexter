Greensock's TweenMax
--------------------
Although not strictly required if you build your own Effects class(es), this library was built with Greensock's TweenMax in mind as the default Tweening engine.  TweenMax is provided in this source tree for convenience.  However, to use TweenMax, you must consent to it's terms of use:

http://www.greensock.com/terms_of_use.html

Use of TweenMax is not covered under the Apache 2.0 license used for the rest of this project.
